### 本地运行
本地执行以下命令即可，注意`FULIBA_USERNAME`和`FULIBA_PASSWORD`需要改成自己的账号密码:
```
cd <this project>
pip install -r requirement.txt
export FULIBA_USERNAME="xxxxxx"
export FULIBA_PASSWORD="xxxxxx"
python3 main.py
```
